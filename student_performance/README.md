# Student Performance Prediction

## Problem Statement

Predict student performance using both regression and classification machine learning models.

## Models Used

### Regression Models

* Linear Regression
* Random Forest Regressor

### Classification Models

* Logistic Regression
* Random Forest Classifier

## Concepts Covered

* Feature engineering
* Pass/Fail prediction
* Feature scaling
* Categorical encoding
* Regression models
* Classification models
* Model evaluation

## Best Models

* Linear Regression performed best for score prediction.
* Logistic Regression performed best for pass/fail classification.
