# Machine Learning Internship Projects

This repository contains machine learning internship projects completed using Python, Pandas, Matplotlib, and Scikit-learn.

## Projects Included

1. Taxi Fare Prediction (Regression)
2. Customer Churn Prediction (Classification)
3. House Price Prediction (Regression)
4. Loan Approval Prediction (Classification)
5. Student Performance Prediction (Regression + Classification)

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Jupyter Notebook

## Machine Learning Concepts Covered

* Data Preprocessing
* Missing Value Handling
* Feature Engineering
* Feature Scaling
* Encoding
* Regression Models
* Classification Models
* Model Evaluation
* Data Visualization
