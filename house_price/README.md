# House Price Prediction

## Problem Statement

Predict house prices using machine learning regression models based on housing-related features.

## Models Used

* Linear Regression
* Ridge Regression
* Random Forest Regressor

## Concepts Covered

* Descriptive statistics
* Feature selection
* Feature scaling
* Data visualization
* Regression models
* Model evaluation

## Best Model

Random Forest Regressor achieved the highest R2 score and best overall performance.
